### What I learned

In this challenge I learned:
How to embedd an SVG in a page using <object> element as a background image.

```html
<object 
  id="dots"
  data="./assets/desktop/bg-pattern-dots.svg" 
  type="image/svg+xml"
  style="width: 232px; height: 104px;">
</object>
```
How to re-ordering the boxes using flexbox order property

```css
.main-visual { /* This is container of form */
    grid-area: content;
    display: flex;
    flex-direction: column;
}

form {
    order: 1;
}
```
